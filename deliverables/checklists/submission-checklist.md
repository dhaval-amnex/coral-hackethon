# Submission Checklist

## Build Integrity
- [ ] Project created during hackathon window.
- [ ] Repo has clear commit history for event period.
- [ ] README includes setup and run instructions.

## Coral Depth
- [ ] Multi-source SQL joins demonstrated.
- [ ] MCP integration demonstrated.
- [ ] Catalog discovery usage shown.
- [ ] Required filter handling shown.

## Business Impact
- [ ] Real enterprise problem statement.
- [ ] Quantified outcome (time saved, quality improved).
- [ ] Clear user persona and workflow.

## Trust and Reliability
- [ ] Security model documented.
- [ ] Token scopes documented.
- [ ] Failure handling documented.
- [ ] Observability evidence included.

## Demo Assets
- [ ] 3-minute live demo script.
- [ ] Backup recorded demo.
- [ ] Architecture diagram.
- [ ] Sample outputs/screenshots.

## Prize Eligibility
- [ ] Starred Coral GitHub repo.
- [ ] Joined Coral Discord.
