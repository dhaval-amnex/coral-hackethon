# Metrics Plan

## Primary Win Metric
- Mean time to produce first actionable incident brief.

## Secondary Metrics
- Query success rate.
- False-positive root cause hypothesis rate.
- Evidence completeness score (0-5 sources/sections).
- Time saved vs manual triage baseline.

## Baseline Measurement
1. Run manual triage for 2 historical incidents.
2. Measure time from alert receipt to first summary posted.
3. Record steps and tools touched.

## Coral-Assisted Measurement
1. Run Incident Captain workflow for same incident types.
2. Measure time to generated brief.
3. Compare quality and completeness.
