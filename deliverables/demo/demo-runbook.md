# Demo Runbook (3 Minutes)

## Pre-Demo Checks (T-20 min)
- [ ] `coral source list` shows all required sources installed.
- [ ] `coral source test <source>` passes for each source.
- [ ] Backup incident ID is ready.
- [ ] Screenshare excludes secrets.

## Live Flow
1. Problem framing (20 sec)
- "Incident triage is fragmented across PagerDuty, GitHub, telemetry, and Slack."

2. Coral discovery proof (30 sec)
- Show catalog discovery and required filters.

3. One workflow execution (70 sec)
- Run incident analysis for `INCIDENT_ID`.
- Show joined evidence in one result flow.

4. Actionable briefing (40 sec)
- Show probable cause, owners, impacted services, and next actions.

5. Outcome metric (20 sec)
- "Reduced first-brief time from X min to Y min."

## Backup Plan
- [ ] Recorded fallback video available.
- [ ] Secondary scenario with sanitized data available.
- [ ] If one source down, run partial-evidence mode.
